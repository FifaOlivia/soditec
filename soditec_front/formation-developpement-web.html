Développement Web

Apprenez à créer des sites dynamiques et interactifs

À propos de cette formation

Cette formation vous permettra de maîtriser les technologies modernes du développement web, y compris HTML, CSS, JavaScript, ainsi que des frameworks comme React et Node.js. Vous apprendrez également à créer des applications web dynamiques et interactives.

Programme de la formation

  • Introduction au développement web
  • HTML et CSS avancés
  • JavaScript et frameworks front-end
  • Back-end avec Node.js et Express
  • Base de données et API REST
  • Projet final

Informations pratiques

Durée : 6 mois

Prérequis : Aucun

Modalités : En ligne ou en présentiel

Prix : 500 €
